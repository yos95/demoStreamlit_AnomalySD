## Informations générales

Le défi DCASE2020 met au défi de trouver un modèle capable de prédire les anomalies générées par des machines d'usine sachant que le jeu de données est essentiellement composé de données audio normales. Le jeu de données dispose de quelques fichiers anormaux dans un jeu de données test pour chacune des 6 machines. Les fichiers de chaque machine sont répartis sur trois ou quatre ID selon la machine.
